# Data Sources

This document provides detailed attribution for the linguistic data used in Sklonuj.

## MorphoDiTa and MorfFlex CZ 2.1

**What it provides:** Czech noun declension forms (all seven cases, singular and plural) generated via the MorphoDiTa REST API.

**Source:** [MorphoDiTa](https://ufal.mff.cuni.cz/morphodita) -- Morphological Dictionary and Tagger, developed at the Institute of Formal and Applied Linguistics (UFAL), Faculty of Mathematics and Physics, Charles University, Prague.

**Underlying dictionary:** [MorfFlex CZ 2.1](https://ufal.mff.cuni.cz/morphodita/users-manual#czech-morfflex2-pdtc), a comprehensive Czech morphological dictionary.

**License:** [CC BY-NC-SA 3.0](https://creativecommons.org/licenses/by-nc-sa/3.0/)

## Wiktionary / kaikki.org

**What it provides:** English translations for Czech nouns.

**Source:** [Wiktionary](https://en.wiktionary.org/) content extracted and structured by [kaikki.org](https://kaikki.org/).

**License:** [CC BY-SA 3.0](https://creativecommons.org/licenses/by-sa/3.0/) (Wiktionary content)

## Data Pipeline

The build pipeline works as follows:

1. A curated word list of Czech nouns is maintained in the repository.
2. For each noun, declension forms across all seven cases (nominative, genitive, dative, accusative, vocative, locative, instrumental) in both singular and plural are generated by querying the MorphoDiTa REST API.
3. English translations are sourced from Wiktionary data dumps processed by kaikki.org.
4. The combined data (lemma, gender, declension forms, translations) is stored and served to the application at runtime.
